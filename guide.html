OTP Viewer Guide

Simple, secure, and powerful two-factor authentication management for your browser.

Manual Entry

Perfect for adding accounts when you only have a secret key without a QR code.

  • Open Settings and stay on Manual Entry.
  • Enter an Account Name (e.g. your email).
  • Enter the Issuer name (e.g. Google).
  • Paste your Secret Key (Base32 format).

Google Import

Transfer all your existing accounts from your mobile device instantly.

  • In Google Authenticator app: Settings → Transfer → Export.
  • Scan the generated QR code with a reader to get the link.
  • Copy the otpauth-migration:// URL.
  • Paste it into the Google Import tab.

GitHub Sync

Automatically back up your data to a private GitHub Project v2.

  • Create a PAT (Token) with the project scope.
  • Create a GitHub Project (v2) and get its number.
  • The data is stored as a Draft Issue named TOTP_BACKUP_DATA.

Data Clean up

Manage your account list and keep it organized with easy deletion.

  • Navigate to the Delete tab in Settings.
  • Check the items you want to remove.
  • Use Select All for quick batch operations.
  • Confirm via our secure custom dialog.
